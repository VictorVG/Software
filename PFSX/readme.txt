PhotoFiltre Studio 10.13.1 Extended Build R1

PhotoFiltre Studio Extended Build - это расширенная сборка, которая
дополнительно к стандартной поддержке плагинов для импорта из форматов RAW и
JPEG 2000, включает в себя ещё и свыше 40 модулей, свыше 200 Photoshop
совместимых бесплатных плагинов, кроме них в состав расширенной версии входят
227 различных дополнительных масок, свыше 230 дополнительных патеров
(шаблонов), свыше 98 дополнительных текстурных шаблонов, 8 рамок для
фотографий, 297 дополнительных контурных масок выбора, 219 кистей, и кроме них
свыше 200 дополнительных фигурных масок выбора, 70 градиентных масок, не
считая мощных средств по работе с изображениями в самой программе.

PhotoFiltre Studio обеспечивает поддержку фильтров от Adobe Photoshop (формата
8bf, только 32 бит! устанавливать в любой каталог, допустимо вложение
подкаталогов) что позволяет дополнительно к имеющимся в дистрибутиве Extended
Build Photoshop совместимым плагинам подключить любое количество нужных
Вам наборов плагинов, расположив их во внешнем, по отношению к основному
каталогу Plugins программы, каталоге (путь к этому каталогу указывается в
"Настройки -> 8BF") и по мере необходимости переключать их, правда при этом
потребуется перезапуск программы PhotoFiltre Studio.

Автор сборки пробовал запускать программу с использованием почти 2000
Photoshop совместимых Plug-ins. Программа запускается быстро, но из-за того,
что меню 8BF плагинов "вылезает" за нижний край экрана, пользоваться ей
становится не удобно. Поэтому было принято решение ограничить число входящих в
сборку плагинов. Для модуля импорта из формата DjVu нужен XnView 1.97.8 или
новее (используется GFL SDK 2.11/3.40). Так же имеется возможность импорта
изображений из XnViewer с помощью дополнительных библиотек, входящих только в
Extended Build (используются средства GFL SDK 2.11/3.40).

Возможность выбора языка интерфейса при установке имеется только в Extended
Build. В текущей редакции Вы можете выбрать при установке только Английский
или Русский язык интерфейса. При необходимости существует техническая
возможность добавления поддержки и других языков в пакет установки в будущих
редакциях Extended Build.

Для выбора языка используется экран выбора компонент, и если там язык не
выбран, то считается, что выбран системный язык. Логика работы автомата
проста: если язык ядра системы отличается от английского или русского и язык
интерфейса не выбран, то ставим с английский язык интерфейса. Для русской
версии Windows по умолчанию при отсутствии выбора ставится русский язык
интерфейса.

По умолчанию все флажки выбора, кроме базовых компонент программы и в том
числе и флажки выбора языка сброшены. Это позволило упростить взаимодействие с
пользователем путём уменьшения числа выводимых диалоговых окон.

Старые плагины совпадающие по именам с новыми и неверно установленные
библиотеки LibGFL SDK удаляются при установке автоматически. Так же при
установке автоматически удаляются языковые модули Studio*.plg т.к. программа
загружает только тот модуль который идёт первым по алфавиту (см. документацию
на сайте). Дополнительно разработчики дополнений могут выбрать установку
Plug-Ins SDK.

Краткое описание программы установки:

Начиная с выпуска 10.3.3 программа установки модернизирована с тем, чтобы
свести к минимуму риск потери или повреждения данных пользователя.

Для этого в программу установки встроены дополнительные диагностические
средства которые автоматически диагностируют наличие запущенных стандартных
установочных процессов msiexec, setup, install, upgrade, update и при их
обнаружении выдают диагностическое сообщение после чего прекращают установку.

Так же детектируется факт запуска программы PhotoFiltre Studio и предлагается
на выбор автоматическое её закрытие (Установка v1.2 и новее сначала ждёт три
секунды закрытия программы и только если она сама не закроется, прекратит её
аварийно) или ручное её завершение. При ручном завершении программы риск
потери данных сводится к минимуму. Дополнительно установочный пакет может
обнаружить и некоторые системные ошибки при наличии которых выдаётся сообщение
и установка автоматически завершается.

Кроме этого программа установки при запуске всегда проверяет имя своего
исполняемого файла с учётом регистра и отсутствие повреждений и изменений
и при их наличии выдаёт диагностическое сообщение после чего автоматически
завершается. Такое же поведение программы будет и при обнаружении системных
ошибок.

При удалении сборка удаляется полностью, включая и те элементы которые
дополнительно установил пользователь. Это сделано специально чтобы в ОС не
осталось мусора.

Дополнительно проверяется наличие в системе установленного Apple QuickTime
7.7.5, и при отсутствии необходимой версии выдаётся сообщение. Дальше Вы
должны выбрать - или продолжить установку, но потом установить QuickTime
отдельно, или прервать её для того, что бы сначала поставить с сервера Apple
QuickTime 7.7.5 или новее (рекомендуется сначала установить QuickTime).

Примечание:

Корпорация Apple официально прекратила разработку QuickTime для ОС Windows.
Последняя версия Apple QuickTime for Windows v7.79.80.95 и если разработчик
официально не объявит о возобновлении разработки Apple QuickTime for Windows,
то любые публикации "новейших версий" Apple QuickTime for Windows следует
считать подделкой и игнорировать. Гарантировать что при их использовании у
Вас не возникнет проблем не возможно, более того, скорее всего они будут,
причём самые неожиданные.

Важно:

Самодельные сборки QuickTime обычно неправильно прописывают параметры и ставят
не все необходимые модули пакета, а по выбору их авторов. Это может вызвать
ошибки в использующих их функции прикладных программах.

Поэтому я рекомендую устанавливать QuickTime скачанный только с сайта его
разработчика. Пока с ним проблем не было, а самодельными сборками - много.

Установка и удаления программы выполняются стандартными средствами системы.
